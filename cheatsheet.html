Ruby Enumerables Cheatsheet
Enumberable methods are tools used to collect, traverse, sort through, and manipulate data stored in various data structures. The two most common enumerable data structures are arrays and hashes.
  • collect: Runs a block on each item in an enumerable and returns it in a new array.

  • count: Returns the number of items in an enumerable, or the number of items that return 'true' for an optionally passed block.

  • cycle: Calls a block for each element n times or forever if none or nil is given.

  • each_with_index: Calls a block with two arguments -- the item and its index -- for each item in enumerable.

  • include?: Returns a boolean value 'true' if any member of enumerable == the object passed.

  • inject: Combines all elements of an enumerable by applying a binary operation that is specified by a block or a symbol that names a method or operator.

  • map: Returns a new array with the results of running block once for every element in the enumerable.

  • reduce: Combines all elements of an enumberable by applying a binary operation, specified by a block or a symbol that names a method or operator.

  • select: Returns an array containing all elements of an enumberable for which the given block returns a true value.

  • sort: Returns an array containing the items in an enumerable sorted, either by value or otherwise if a specific block is given.

collect { |obj| block } --> array

count --> int
count(item) --> int
count { |obj| block } --> int

cycle(n=nil) { |obj| block } --> nil
cycle(n=nil) --> an_enumerator

each_with_index(*args) { |obj, i| block } --> enum
each_with_index(*args) --> an_enumerator

include?(obj) --> true or false

inject(initial, sym) --> obj
inject(sym) --> obj
inject(initial) { |memo, obj| block } --> obj
inject { |memo, obj| block } --> obj

map { |obj| block } --> array
map --> an_enumerator

reduce(initial, sym) --> obj
reduce(sym) --> obj
reduce(initial) { |memo, obj| block } --> obj
reduce { |memo, obj| block } --> obj

select { |obj| block } --> array
select --> an_enumerator

sort --> array
sort { |a, b| block } --> array
